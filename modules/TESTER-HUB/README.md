TESTER-HUB — тестовый центр JD7 WORLD

TESTER-HUB — это модуль JD7 WORLD, который объединяет:

- роли тестировщиков (Tester.001, Tester.002, ...),
- сценарии тестирования (включая Free JD7),
- чек-листы и стандарты отчётности,
- ссылки на шаблоны задач в .github/ISSUE_TEMPLATE.

Цели

- Создать единый вход для всех, кто тестирует JD7.
- Обеспечить повторяемость и прозрачность тестов.
- Связать реальные сценарии с задачами в репозитории.

Структура

- scenarios/ — описания сценариев (Free JD7 и другие модули).
- checklists/ — чек-листы по типам тестов (функциональные, UX, регрессия).
- reports/ — примеры отчётов, формат сводок.

Роли

- Tester.001 — базовое функциональное тестирование.
- Tester.002 — углублённое тестирование фич и UX.
- Tester.N — специализированные роли (нагрузка, безопасность, интеграции).

Каждая роль привязана к своим шаблонам задач и чек-листам.
